# Job Search Project

## Overview
The Job Search project is designed to help users find job opportunities efficiently. It utilizes various APIs to fetch job listings and provides a user-friendly interface for searching and applying for jobs.

## Features
- Integration with USAJOBS API to fetch job listings.
- Integration with Gemini API for additional functionalities.
- User-friendly search interface.

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd Job Search
   ```
3. Install the required dependencies:
   ```
   npm install
   ```

## Configuration
Before running the project, you need to set up your environment variables. Create a `.env` file in the `utils` directory and add your API keys:

```
USAJOBS_API_KEY = "your_usajobs_api_key"
GEMINI_API_KEY = "your_gemini_api_key"
```

## Usage
To start the application, run the following command:
```
npm start
```
